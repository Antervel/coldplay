## T0003 — Implement https://github.com/dvadell/extra_credo
git branch: `ai-agent/t0003_-_implement_extra_credo

**Problem:**
We would like to have more credo rules. Implement the rules in https://github.com/dvadell/extra_credo

**Solution:**
Add the `extra_credo` package as a dependency and register all 19 custom Credo checks (NoFloatForMoney, NoBareChangesetError, NoAssignNewInMount, NoAuthInHandleEvent, NoExternalResource, NoImplicitCrossJoin, NoPubsubWithoutConnected, NoRawUntrusted, ObanAtomKeys, ObanStructInArgs, UnpinnedQueryBindings, NoColorfulEmoji, NoCommentsAsCommitMessages, NoDbQueryInMount, NoDedupBeforeCastAssoc, NoDirectThirdPartyCalls, NoLocaleInTaskClosure, NoNonIdempotentJobs, NoUnsupervisedProcesses) in `.credo.exs`. The upstream package had a bug where issues were created without the `:check` field, causing a `nil.id()` crash. The fix involves a locally patched copy of the repo that adds `check: __MODULE__` to all issue structs.
